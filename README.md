Welcome to the official support page for HotdogUp.
If you are experiencing issues or have questions, please use the resources below.

🛠 Troubleshooting First Steps
Before reaching out, please try the following:

Update the App: Ensure you are running the latest version from the App Store.

Restart: Close the app completely and restart your device.

Check Connection: Ensure you have a stable internet connection if the app requires data.

🐞 Reporting a Bug
If you've found a technical issue, please open a GitHub Issue in this repo. Please include:

Your iOS Version (e.g., iOS 19.x)

Your Device Model (e.g., iPhone 16 Pro)

Steps to reproduce the bug.
